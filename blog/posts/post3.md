VS Code is my go-to editor, and extensions make it even more powerful. Here are my must-have extensions for 2025.

## Productivity Extensions

- **Error Lens:** Inline error messages
- **GitLens:** Supercharge Git capabilities
- **Todo Tree:** Manage TODO comments
- **Bookmarks:** Navigate code quickly

## Language-Specific

Depending on what you're working on:

- **Python:** Pylance, Ruff
- **Web:** ESLint, Prettier, Live Server
- **Markdown:** Markdown All in One

## Theme & UI

Make your editor look good:

- **One Dark Pro** - My favorite dark theme
- **Material Icon Theme** - Beautiful icons
- **Bracket Pair Colorizer** - Color-matched brackets
